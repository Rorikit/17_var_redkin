##Конвертер учебного конфигурационного языка в JSON

Инструмент командной строки для преобразования текста на учебном конфигурационном языке в формат JSON. Разработан в соответствии с вариантом №17 задания.
Особенности

    Полная поддержка синтаксиса учебного конфигурационного языка

    Строгая проверка соответствия спецификации

    Рекурсивный парсинг вложенных структур

    Подстановка переменных в строках и значениях

    Детальные сообщения об ошибках

    Комплексное тестирование

Установка и запуск
Требования

    Python 3.6 или выше

    Внешние зависимости не требуются

Быстрый старт
bash

# Клонирование репозитория
git clone <repository-url>
cd config-converter

# Конвертация конфигурационного файла
python main.py -i examples/webserver.config

# Запуск тестов
python -m unittest test_parser.py

Использование
Базовое использование

python main.py -i путь/к/файлу.config


Результат выводится в стандартный вывод в формате JSON.


    -i, --input - путь к входному файлу (обязательно)
